## About

PetPal is a web-based quiz application that helps users determine which pet best suits their lifestyle. By answering questions about activity level, home environment, time availability, and pet preferences, users receive a personalized pet recommendation along with a care guide.

## Features

- Interactive quiz with multiple choice questions
- Calculates best pet match using a scoring system
- Handles ties with a random selection
- Displays a detailed pet care guide including:
  - Nutrition
  - Exercise
  - Healthcare
  - Grooming
- Images for each pet
- Responsive design for mobile, tablet, and desktop

## How to Use

1. Clone or download the repository:
   ```bash
   git clone https://github.com/YOUR-USERNAME/petpal-quiz.git
   ```
2. Open `index.html` in your browser.
3. Take the quiz by answering all the questions.
4. View your personalized pet match and care guide.
